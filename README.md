# Speaker Identification in Noisy Environment

## Project Overview

This project presents an efficient speaker identification system capable of recognizing speakers in noisy environments using speech processing and machine learning techniques.

## Technologies Used

- Python
- Librosa
- NumPy
- Scikit-Learn
- HMMLearn
- Matplotlib

## Methodology

1. Audio Input
2. Noise Addition (White Gaussian Noise)
3. Spectral Subtraction
4. MFCC Feature Extraction
5. CMVN Normalization
6. Gaussian HMM Training
7. Speaker Prediction
8. Performance Evaluation

## Dataset

The dataset contains speech recordings from 18 speakers.

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix

## Results

The system demonstrates effective speaker identification performance across various noisy conditions and Signal-to-Noise Ratio (SNR) levels.
